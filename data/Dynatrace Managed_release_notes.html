New features and enhancements

Root cause for cluster events

Cluster events related to transaction storage truncation now present the specific root cause - insufficient disk space or a disk quota limit.

Configuration changes in audit log

Cluster configuration changes based on Cluster API v2 Settings are now also included in the Audit Log UI.

New opt out options

You can now opt out from sending user names, user email addresses, and associated permissions to Mission Control. By opting out, you block access to Dynatrace support services to manage support tickets.
